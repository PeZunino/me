# pedrozunino.com

Meu site pessoal — desenvolvido com Next.js e TypeScript, hospedado na Vercel.

🔗 **[pedrozunino.com](https://www.pedrozunino.com)**

---

## Sobre

Site minimalista com foco em apresentação profissional. Conta um pouco sobre minha trajetória, lista minhas experiências na área de tecnologia e oferece links para contato.

## Tech Stack

- **Framework:** Next.js (App Router)
- **Linguagem:** TypeScript
- **Estilo:** CSS / SCSS
- **Deploy:** Vercel

## Rodando localmente

```bash
# Instalar dependências
pnpm install

# Iniciar servidor de desenvolvimento
pnpm dev
```

Acesse [http://localhost:3000](http://localhost:3000) no navegador.

## Estrutura

```
src/        # Código-fonte da aplicação
public/     # Assets estáticos
experiences.json  # Histórico profissional em JSON
```
